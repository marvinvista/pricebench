# Security

Pricebench is a local CLI. It does not call external services, store credentials, or send pricing models over the network.

## Reporting

Please report security issues privately by opening a private advisory on GitHub or emailing the maintainer listed in `package.json`.

Include:

- A concise description of the issue.
- Steps to reproduce.
- Affected version or commit.
- Expected impact.

## Supported Versions

Until `1.0`, security fixes target the latest published minor release and the current `main` branch.
